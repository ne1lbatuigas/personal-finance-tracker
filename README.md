# Personal Finance Tracker

A simple, modern web app to track your personal finances. Add, edit, and delete income or expense entries, visualize your data with a pie chart, and export your records as CSV. Supports dark and light mode toggle for a comfortable user experience.

## Features
- Add, edit, and delete transactions (income/expense)
- Data persists in your browser (localStorage)
- Pie chart visualization (Chart.js)
- Export data as CSV
- Responsive design
- Dark mode and light mode toggle

## Usage
1. Open `index.html` in your browser.
2. Add your transactions using the form.
3. Edit or delete entries using the table's action buttons.
4. Switch between dark and light mode using the toggle button.
5. Export your data as CSV with the export button.

## Project Structure
- `index.html` – Main HTML file
- `style.css` – App styling (dark/light mode)
- `script.js` – App logic and chart rendering

## Dependencies
- [Chart.js](https://www.chartjs.org/) (via CDN)
